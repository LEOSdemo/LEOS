Problem Statement

"Drafting EU legislation is a complicated process, even more when many stakeholder are involved, information is presented in both digital and physical styles, and one has to deal with versioning documents."

What LEOS is

LEOS is an authoring tool for EU legislation with an extensible backend code that allow users to add comments, suggestions and co-editions in order to ease the management and understanding of EU legislation. However, LEOS must not be confused with a word processor or a backend application for national legislation.

Collaboration

Losing time exchanging and managing different versions with your draſting partners? LEOS is built as an online authoring tool, a single online workspace for all contributors, offering simultaneous collaboration on one draſt.

Import

If you need to reuse some text from existing sources like the Official Journal of the European Union, you can use the import feature. You can import some or all recitals and articles of an act.

Structure

If the structure is your challenge, our solution deals with the creation of acts from a template, and has all further editions subject to predefined structure rules.
Additionally, there is no need to worry about the internal references, they are automatically created and updated.

Review & Commenting

Features for review, comments or suggestions which can be approved (or rejected) and merged directly in the text are also available. The solution includes also the track changes option

Rich text

If the text needs to be enriched with images, tables and mathematical formula, this is also possible. The rich text can be added as-is and can be customized by adding plugins.

Versioning

All versions are centrally stored and protected using role based access control rules. With this feature, the users can have instant comparisons between successive versions and the timeline is visually displayed and easy to manage
